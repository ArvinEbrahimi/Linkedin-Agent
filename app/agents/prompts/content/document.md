# Document Outline Template

Create a LinkedIn **long-form document / article outline** (native document post).

## Format Requirements
- **Title:** Keyword-rich, benefit-driven
- **Introduction:** Hook + problem statement (3-4 lines)
- **Sections:** 4-6 H2 sections with bullet key points
- **Conclusion:** Summary + CTA
- Provide the full outline as `full_post` with clear section headers

## Also Provide
- Exactly **3 title/hook variants**
- 3–5 hashtags
- First comment suggestion
- 2–3 alternative document angles

## 2026 Optimization
- Deep value content drives saves and profile visits
- Include one quantified result or case study reference
