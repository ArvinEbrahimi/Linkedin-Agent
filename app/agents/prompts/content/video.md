# Short Video Script Template

Create a LinkedIn **short video script** (30–90 seconds — +36% growth in 2026).

## Format Requirements
- **[0-3s] Hook:** Visual + verbal pattern interrupt
- **[3-30s] Story/Value:** Core message with one proof point
- **[30-60s] Deep value:** Actionable tip or insight
- **[60-90s] CTA:** Clear next step
- Include on-screen text suggestions in brackets: [TEXT: ...]
- Note B-roll or screen recording ideas where helpful

## Also Provide
- Exactly **3 hook variants** for the opening 3 seconds
- 3–5 hashtags
- First comment suggestion (for links)
- 2–3 alternative video angles

## Iranian User Note
- If bandwidth is limited, suggest text-heavy video with minimal B-roll
- Subtitles are mandatory (many watch muted)
