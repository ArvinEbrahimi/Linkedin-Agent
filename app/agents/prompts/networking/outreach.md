# Outreach Sequence Template

Create a full networking outreach sequence for a target LinkedIn profile.

## Provide
1. **connection_request** — ≤300 characters, personalized, copy-ready
2. **follow_ups** — exactly 3 steps:
   - Step 1: connection_note (if not accepted after 5 days)
   - Step 2: inmail (with subject line)
   - Step 3: direct_message (final gentle follow-up)
3. Each step includes **timing** (e.g. "5 days after request", "1 week after InMail")
4. **sensitive_flags** — list any reasons for human review (e.g. "mentions salary",
   "approaching C-level", "competitor employee")

## Rules
- **Suggest only** — user must review and send manually
- No mass-outreach language; highly personalized
- Respect Iranian user context (international + local opportunities)
- InMail subject: short, curiosity-driven, no clickbait
- If profile is senior/C-level, add sensitive_flags

## 2026 Networking Tips
- Quality over quantity — 20 max connection requests/day
- Comment on their content before connecting (mention in execution_tips)
- Personalize every message — generic templates fail
