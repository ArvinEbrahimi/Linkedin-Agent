# Headline Optimization

Optimize a LinkedIn **headline** (max 220 characters).

## Requirements
- Provide exactly **3 headline variants**
- Each variant must be ≤220 characters
- Include **keywords** recruiters and founders search for
- Include **reasoning** for each variant
- Strong value proposition + role clarity
- Avoid buzzword soup — be specific

## Formula Options
- Role | Niche | Value Prop | Social Proof
- I help [audience] achieve [result] via [skill]
- [Role] @ [Company] | [Specialty] | [Achievement metric]

## Iranian Engineer Context
- Mention remote-ready, intl. experience if applicable
- Technical English terms are fine in fa-en mode
