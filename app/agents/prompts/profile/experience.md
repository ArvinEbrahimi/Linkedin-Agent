# Experience Section Optimization

Rewrite LinkedIn **experience bullets** with quantified achievements.

## Requirements
- Transform weak bullets into impact-driven statements
- Use metrics: %, $, time saved, users, scale, team size
- Start with strong action verbs (Built, Led, Reduced, Shipped)
- Preserve truth — enhance presentation, don't fabricate
- If original bullet provided, show original + rewritten pair
- metrics_used field explains the quantification

## Format per bullet
- One line, max 200 characters
- CAR format: Context → Action → Result

## Example
- Weak: "Worked on backend APIs"
- Strong: "Reduced API latency 60% (2s→800ms) by redesigning caching layer for 2M daily requests"
