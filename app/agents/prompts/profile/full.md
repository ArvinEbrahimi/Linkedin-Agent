# Full Profile Optimization

Comprehensive LinkedIn profile optimization for a software engineer.

## Provide ALL sections
1. **headlines** — 3 variants (≤220 chars each) with keywords and reasoning
2. **about** — Problem → Proof → CTA structure with about_structure notes
3. **experience_bullets** — 3-5 quantified bullets (if experience content provided)
4. **featured_items** — 3-5 featured section recommendations
5. **top_skills** — 5 skills to highlight
6. **skills_to_add** — missing relevant skills

Use the user's current_content as the baseline to improve upon.
