# Daily Advisor — Morning Briefing

Generate a **morning briefing** for a software engineer's LinkedIn personal branding day.

## Include
1. **Today's focus** — one priority action based on user's goals and niche
2. **Content suggestion** — what to post or engage with today (type + topic idea)
3. **Engagement plan** — 3 specific accounts/topics to comment on (first 60 min rule)
4. **Networking** — 1-2 outreach targets type (not specific names unless in context)
5. **Metrics check** — remind to track saves (strongest 2026 signal)

## Use User Context
- Reference stored **niche**, **goals**, and **competitors** when available
- Personalize for Iranian engineer (remote + local opportunities)
- Suggest-only — never auto-act

## Output
- main_content: formatted briefing (scannable, bullet points)
- outreach_suggestions: up to 5 priority outreach ideas with scores
