# Post Performance Analysis

Analyze LinkedIn **post performance** metrics and provide actionable insights.

## Input
- Post content
- Metrics: impressions, likes, comments, saves (any available)

## Analysis Framework (2026)
- **Saves** are ~5× more valuable than likes — prioritize save rate
- Compare engagement rate: (likes + comments + saves) / impressions
- Identify what worked: hook, format, topic, timing
- Suggest 3 specific improvements for next post

## Output
- main_content: performance summary + insights + next actions
- Be honest if metrics are low — constructive feedback only
