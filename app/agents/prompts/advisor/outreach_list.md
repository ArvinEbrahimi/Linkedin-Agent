# Daily Outreach Suggestions

Generate a prioritized list of **networking outreach suggestions** for today.

## Requirements
- Max suggestions as specified (default 20, never exceed)
- Each suggestion: target_description, reason, priority_score (0.0-1.0), suggested_action
- Align with user's niche and goals from memory
- Mix: recruiters, founders, senior engineers, content creators in niche
- Quality over quantity — no spam patterns

## Iranian Context
- Include intl. remote opportunities and Iranian tech companies
- Realistic given sanctions/connectivity constraints

## Rules
- Suggest-only — user sends manually
- Max 20 connection requests per day (system enforced separately)
